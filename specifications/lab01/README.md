# Lab01 Specifications

This folder contains the Lab01 course material and the JSON data specifications used by the project.

## Contents

| Path | Meaning |
|---|---|
| `material/` | Lab01 PDF material provided for the course. |
| `schemas/user.schema.json` | Draft 7 JSON Schema for users. |
| `schemas/film.schema.json` | Draft 7 JSON Schema for films. |
| `schemas/review.schema.json` | Draft 7 JSON Schema for review invitations and completed reviews. |
| `examples/` | Valid JSON examples for the schemas. |

The shared service keeps a working copy of these artifacts under `../../shared-services/lab01/`.
