# Documentation

This directory contains the written analysis for the project.

## Contents

- `01-swaggerhub-analysis.md`: documents the initial SwaggerHub / Swagger Studio analysis, including available features, trial-account limitations, generated project structure, and early conclusions.
- `02-experimental-comparison.md`: documents the runtime experiment, regeneration behavior, adapter strategy, patch scripts, and comparison between the OpenAPI Generator and SwaggerHub workflows.

These files explain the reasoning behind the final project structure and support the experimental conclusions.
