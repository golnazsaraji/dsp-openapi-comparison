# Generated SwaggerHub Services

This directory contains service-layer files generated by SwaggerHub / Swagger Codegen.

## Contents

- `DefaultService.js`: generated service file patched to delegate to `adapters/swaggerhub/DefaultServiceAdapter.js`.

This directory is important for the regeneration experiment because `DefaultService.js` is overwritten when the server stub is regenerated. The script `../../scripts/patch-swaggerhub.js` restores the adapter connection after regeneration.
