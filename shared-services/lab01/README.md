# Lab01 Design Artifacts

This directory keeps the Lab01 design material inside the shared service area of the final project.

## Contents

| Path | Meaning |
|---|---|
| `schemas/user.schema.json` | Draft 7 JSON Schema for the Lab01 user data structure. |
| `schemas/film.schema.json` | Draft 7 JSON Schema for the Lab01 film data structure. |
| `schemas/review.schema.json` | Draft 7 JSON Schema for the Lab01 review invitation data structure. |
| `examples/` | Valid JSON examples used to check the schemas. |

The canonical OpenAPI contract for the final Film Manager service is still stored in `../../openapi/openapi.yaml`. These schemas document the first Lab01 design step and the data rules that the service contract is based on.
