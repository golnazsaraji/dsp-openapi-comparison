# Generated OpenAPI Services

This directory contains service-layer files generated by OpenAPI Generator.

## Contents

- `DefaultService.js`: generated service file that delegates to `adapters/openapi-generator/DefaultServiceAdapter.js`.
- `Service.js`: generated helper for building success and error responses.
- `index.js`: generated service export file.

This directory represents the earlier patch-based OpenAPI Generator experiment. The final custom generation workflow uses the templates in `../../out/` to create adapter delegation automatically.
