# Generated OpenAPI Services

This directory contains service-layer files generated by OpenAPI Generator.

## Contents

- `DefaultService.js`: generated service file patched to delegate to `adapters/openapi-generator/DefaultServiceAdapter.js`.
- `Service.js`: generated helper for building success and error responses.
- `index.js`: generated service export file.

This directory is important for the regeneration experiment because `DefaultService.js` is overwritten when the server stub is regenerated. The script `../../scripts/patch-openapi-generator.js` restores the adapter connection after regeneration.
