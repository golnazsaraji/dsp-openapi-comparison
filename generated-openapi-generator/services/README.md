# Generated OpenAPI Services

This directory contains service-layer files generated by OpenAPI Generator.

## Contents

- `DefaultService.js`: generated service file patched to delegate to `../../adapters/initial-example/DefaultServiceAdapter.js`.
- `Service.js`: generated helper for building success and error responses.
- `index.js`: generated service export file.

This directory represents the restored initial simple example. The final custom generation workflow uses the templates in `../../out/` to create adapter delegation automatically for `../generated-openapi-generator-custom/`.
